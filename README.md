# 🏠 Acerca del proyecto

**Rental Agreement Management Contract** es una solución blockchain orientada a la administración de contratos de alquiler mediante tecnología descentralizada y Smart Contracts en Ethereum.

El sistema fue desarrollado para:

- 📝 Gestionar contratos de renta
- 💰 Automatizar pagos mensuales
- 🔐 Garantizar transparencia e inmutabilidad
- ⚡ Reducir disputas entre arrendador e inquilino
- 📊 Registrar operaciones on-chain
- 🏛️ Eliminar intermediarios
- 🌐 Implementar soluciones Web3 descentralizadas

La plataforma utiliza contratos inteligentes para asegurar procesos confiables y auditables dentro del ecosistema blockchain.

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/usuario/rental-agreement-management-contract.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd RentalAgreementManagementContract
```

---

## 3️⃣ Instalar dependencias

```bash
npm install
```

---

## 4️⃣ Configurar red blockchain

```env
PRIVATE_KEY=
RPC_URL=
ETHERSCAN_API_KEY=
```

---

## 5️⃣ Compilar contratos

```bash
npx hardhat compile
```

---

## 6️⃣ Desplegar Smart Contract

```bash
npx hardhat run scripts/deploy.js --network sepolia
```

---

## 7️⃣ Ejecutar aplicación

```bash
npm run dev
```
